# Bass Neck Note Quiz

A React/Vite app for learning the notes on a 4-string bass neck.

## Features

- 20-question pop quiz
- 3-second countdown before the first question
- 1-second pause between questions
- Timer options: 8, 6, or 4 seconds per question
- User-selected note set for each run
- Questions restricted to the selected note names
- Selected notes shown in different positions across the bass neck
- Standard EADG bass tuning
- Frets 0-12
- One highlighted fret per question
- Multiple-choice note buttons
- Final score and elapsed time
- Persistent top 3 most often wrong note positions
- End-of-quiz answer review with labelled fretboard diagrams
- Cell-anchored neck-style fret markers at 3, 5, 7, 9, and double-dot 12. Markers are rendered inside the actual fret cells so they stay aligned with the fret numbers and highlighted note.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
